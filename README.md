# 小哈电量小组件


### 使用方式

#### IOS
0. AppStore 中下载 Scriptable
1. 复制代码到 Scriptable 中
2. 修改 batteryNo 为你的小哈租电里面的电池号码

<img src="./images/demo.png" width="300" />

#### Android

0. 下载 Release 中的 apk
1. 修改 batteryNo 为你的小哈租电里面的电池号码
